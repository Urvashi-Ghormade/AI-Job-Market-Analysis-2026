# AI Career Insights & Salary Analytics Dashboard (2026)

## 📌 Project Overview
This project provides an **End-to-End Analysis** of the AI and Data Science job market in 2026. The primary goal is to visualize current demand, compensation benchmarks, and evolving work-culture trends (Remote vs. On-site) within the AI industry. The dashboard is designed following Business Intelligence best practices to provide both high-level insights and granular data details.

## 🚀 Key Features
* **Executive Summary Dashboard:** A high-level overview featuring key performance indicators (KPIs) such as Total Job Openings, Average Annual Salary, and Most In-Demand Roles.
* **Advanced Visualizations:** Utilization of Gauge charts, Ribbon charts, and Scatter plots to identify correlations between experience levels and compensation.
* **Granular Data View:** A dedicated detailed report page incorporating **Conditional Formatting** (Data Bars) to highlight salary variations and skill distributions across rows.
* **Seamless Navigation:** Integration of custom navigation buttons and "Back" functionality for an intuitive user experience.

## 🛠️ Tools & Technologies Used
* **Power BI Desktop:** Used for dashboard architecture and visual storytelling.
* **DAX (Data Analysis Expressions):** Implemented for custom measures and complex data calculations.
* **Power Query:** Leveraged for ETL (Extract, Transform, Load) processes and data cleaning.
* **Data Modeling:** Established relationships between job roles, salary scales, and geographical work models.

## 📊 Key Insights
* **In-Demand Expertise:** The analysis identifies **AI Engineer** as the most sought-after role in the current market.
* **Salary Benchmarking:** The average annual salary for specialized AI roles has reached a benchmark of **$113K**.
* **Work Model Dynamics:** There is a significant shift towards **Hybrid and Remote** models, specifically for Mid-to-Senior level positions.

## 📁 Repository Structure
* `AI_Trend_Analytics_2026.pbix`: The core Power BI Dashboard file.
* `AI_Market_Detailed_Report.pdf`: A print-friendly PDF version of the dashboard and detailed data tables.
* `Dataset/`: # AI Career Insights & Salary Analytics Dashboard (2026)
